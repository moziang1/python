##### 临时使用pip清华镜像源

```
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple/ 包名
```

永久使用pip清华镜像源

```
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple/
```

#升级pip

```
python -m pip install --upgrade pip  升级pip的最新版本
```
